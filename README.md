# 🎬SISTEMA DE LOCAÇÃO DE FILMES - ENACFLIX (JDBC)
---
#### Este projeto visa o desenvolvimento de um sistema simplificado e prático de locadora de filmes utilizando java.
 ---
 ## ⚙️Ferramentas
 - sistema completo com funções de cadastrar filmes, generos, listar, alocar e muito mais...
 - registro de usuário
 - gerenciamento de senhas com hash para maior segurança do usuário
 - exportação e importação de dados
 ---
 ## 🧑‍💻Tecnologias
 - Java
 - MySQL
 --- 
 ## 📊 Estrutura do Banco de Dados
 - filmes: Armazena título, ano, faixa etária e status.
 - gêneros: armazena gêneros.
 - empréstimos: armazena locações e alocações.
 - Tabela Usuários: Gerenciamento de credenciais administrativas.
 ---
## 🚀 Como utilizar
- 1- Importe o banco de dados utilizando o script SQL fornecido.
- 2- Adicione a biblioteca mysql-connector-j ao seu projeto no NetBeans.
- 3- Configure as credenciais de acesso no arquivo de conexão.
- 4- Execute a classe Locadora.java para iniciar o sistema.
